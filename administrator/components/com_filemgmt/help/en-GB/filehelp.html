Description

A folder named 'dwuploads' will be created into your project root directory. Please give permissions to this folder as all the files will be uploded in this folder. If a folder is not created you need to create one folder named 'dwuploads', or this component will not work.

Please give the link for the file to download from frontend as per the link formed, you can get the path when you right click on the file name and click option 'Copy Link Location'.
